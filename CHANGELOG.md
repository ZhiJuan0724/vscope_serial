# Changelog

>所有值得用户关注的变更都记录在这里
>提交涉及发布内容时，提交前必须让用户核对本文件对应版本段落
>一个版本tag被推送后再修改，则在新版本日志日志下填写，日期在新版本推送前待定，在确认推送前填上正确的时间

## v1.0.5 - 2026-xx-xx

### Added
- 地址配置支持 CSV 导入，CSV 两列分别为通道名称和通道地址，并自动识别首行是否为表头。
- 地址配置选择支持搜索。
- 开启最新点跟随后拖动画布自动关闭跟随功能

### Changed
- 重新开始绘图或导入数据后仅保留垂直光标开关，X-X、Y-Y、观察和统计等临时工具会自动关闭。
- 原始数据 HEX 发送的 CRC 字节顺序由“高低位反转”改为明确选择大端或小端。

### Fixed
- 修复应用信息中的构建时间可能显示不准确的问题。
- 修复众邦 C 配置文件在部分中文注释和仓库示例下识别不稳定的问题。
- 修复通道名编辑后直接点击通道地址时焦点不会正确失去的问题。
- 修复开启最新点跟随后拖动画布不会自动关闭跟随的问题。
- 修复绘图协议发送二进制数据时在原始数据界面可能按文本显示的问题，并标注为绘图发送。
- 修复停止绘图后偶发无法修改通道地址的问题。
- 优化重复点击绘图按钮时的提示显示，避免提示框堆积并缩短状态提示停留时间。
- 优化多通道偏置模式下的右侧刻度列宽，避免长数值挤占相邻通道显示。
- 应用信息页新增高级设置入口，可关闭应用内临时提示信息。

## v1.0.4 - 2026-05-31

### Added
- 新增应用信息入口，显示应用名称、版本和构建时间。
- 新增手动检查更新，并支持启动时自动检查更新开关。
- 更新检查优先使用 GitHub Release，失败后尝试 Gitee Release。

### Changed
- 完善固定帧协议配置，支持帧头、帧尾、CRC 和逐通道数据类型设置。
- 优化固定帧配置下拉框布局和通道编辑弹窗下拉菜单显示。
- 通道列表在无发送协议和 r 协议之间切换时保持更一致的行高和宽度。

### Fixed
- 修复固定帧 CRC 位置、大小端和逐通道类型组合下的解析问题。

## v1.0.3 - 2026-05-31

### Added
- 拆分绘图接收协议和发送协议，新增 r 协议发送支持。
- 支持旧版虚拟示波器 DAT 导入和众邦 C 文件配置识别。

### Changed
- 优化应用字体和绘图工具栏布局。
- 优化随机数据绘图、导入弹窗和串口断线检测。
