# Changelog

所有值得用户关注的变更都记录在这里。

提交涉及发布内容时，提交前必须让用户核对本文件对应版本段落。

## v1.0.4 - 2026-05-31

### Added
- 新增应用信息入口，显示应用名称、版本和构建时间。
- 新增手动检查更新，并支持启动时自动检查更新开关。
- 更新检查优先使用 GitHub Release，失败后尝试 Gitee Release。

### Changed
- 完善固定帧协议配置，支持帧头、帧尾、CRC 和逐通道数据类型设置。
- 优化固定帧配置下拉框布局和通道编辑弹窗下拉菜单显示。
- 通道列表在无发送协议和 r 协议之间切换时保持更一致的行高和宽度。

### Fixed
- 修复固定帧 CRC 位置、大小端和逐通道类型组合下的解析问题。

## v1.0.3 - 2026-05-31

### Added
- 拆分绘图接收协议和发送协议，新增 r 协议发送支持。
- 支持旧版虚拟示波器 DAT 导入和众邦 C 文件配置识别。

### Changed
- 优化应用字体和绘图工具栏布局。
- 优化随机数据绘图、导入弹窗和串口断线检测。
